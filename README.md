# Third-Person-Inventory-Demo

Демо‑проект: персонаж от третьего лица и инвентарь на UI Toolkit с подбором предметов из мира.

## Требования
- Unity 6+
- Input System (пакет Unity)

## Запуск
1. Откройте проект в Unity 6+.
2. Откройте сцену `Assets/Scenes/Gameplay.unity`.
3. Нажмите Play.

## Управление
- `WASD` — движение
- `Mouse` — вращение камеры
- `E` — подобрать предмет
- `Tab` — открыть/закрыть инвентарь
- `RMB` — выбросить предмет из слота
- Drag & Drop — перемещение предметов между слотами

## Данные предметов
- JSON: `Assets/StreamingAssets/items.json`
- Поля: `id`, `name`, `color`, `stackable`, `maxStack`

## Редактирование базы предметов
Для удобного добавления и правки предметов используйте окно редактора:
1. В Unity откройте `Tools/Item Database Editor`.
2. В блоке **Add New Item** заполните `Id`, `Name`, `Color`, `Stackable`, `Max Stack`.
3. Нажмите **Add Item**, затем **Save**.

Окно редактирует файл `Assets/StreamingAssets/items.json`. При необходимости можно править его вручную.

## Примечания
- Предметы на сцене — простые меши с цветом из JSON.
- Подсказка "Нажмите E" появляется при приближении к предмету.
- Чувствительность вращения настраивается в `PlayerInput` через поле `_lookSensitivity` (используется и камерой, и поворотом персонажа).
